# Requisitos

Tener instalado Node. El ultimo LTS o ultima versión

## Instalar dependencias

```bash
npm i
```

Correr en modo de desarrollo

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Correr en modo normal

```bash
npm run build
npm start
```
